IGListItemController

@interface IGListItemController : NSObject

The base class for item controllers used in the list infra. This class is meant to be subclassed.

  • viewController

    The view controller housing the adapter that created this item controller.

    @discussion Use this view controller to push, pop, present, or do other custom transitions. It is considered very bad practice to cast this to a known view controller and call methods on it other than for navigations and transitions.
