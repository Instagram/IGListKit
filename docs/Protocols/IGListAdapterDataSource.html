IGListAdapterDataSource

@protocol IGListAdapterDataSource <NSObject>

Implement this protocol to provide data to power an IGListAdapter feed.

  • -itemsForListAdapter:

    Asks the data source for an array of items for each list in your feed.

    Declaration

    Objective-C

    - (nonnull NSArray<id> *)itemsForListAdapter:
    (nonnull IGListAdapter *)listAdapter;

    Swift

    func itemsForListAdapter(listAdapter: IGListAdapter!) -> AnyObject!

    Parameters

    listAdapter

    The list adapter requesting this information.

    Return Value

    An array of items for each item in the feed.

  • -listAdapter:itemControllerForItem:

    Asks the data source for an item controller for the specified data item.

    @discussion New list items should be initialized here for items when asked. You are free to inject the item into the list, as there are no other ways for a list to query its item beyond this point. You may also pass any other information to the list at this time.

    List items are initialized for all items whenever the IGListAdapter is created or asked to reload. If the order of an item has changed, the list will be reused.

    Declaration

    Objective-C

    - (nonnull IGListItemController<IGListItemType> *)
          listAdapter:(nonnull IGListAdapter *)listAdapter
itemControllerForItem:(nonnull id)item;

    Swift

    func listAdapter(listAdapter: IGListAdapter!, itemControllerForItem item: AnyObject!) -> IGListItemController!

    Parameters

    listAdapter

    The list adapter requesting this information.
    item

    An item in the feed, provided in -itemsForListAdapter:. You can also pass this to your list.

    Return Value

    An IGListItemType conforming item that can be displayed in the feed.

  • -emptyViewForListAdapter:

    Asks the data source for a view to use as the collection view background when there are no items.

    @discussion This method is called every time the list adapter is updated. You are free to return new views every time, but for performance reasons you may want to retain your own view and return it here. The infra is only responsible for adding the background view and maintaining its visibility.

    Declaration

    Objective-C

    Swift

    func emptyViewForListAdapter(listAdapter: IGListAdapter!) -> AnyObject!

    Parameters

    listAdapter

    The list adapter requesting this information.

    Return Value

    A view to use as the collection view background, or nil if you don’t want a background view.
